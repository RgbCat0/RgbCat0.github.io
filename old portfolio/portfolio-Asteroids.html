Controls:

  • WASD to move
  • Left mouse button to shoot
  • Z to lock on target
  • X to deselect
Description:

Asteroids 3D is one of the first games I made. It is more of a prototype but was a lot of fun to make. The game is a 3D version of the classic Asteroids game. The player controls a spaceship and has to destroy asteroids to get points. The game has a simple scoring system and has a bullet prediction system. The game is made in Unity and is available to play on itch.io.
